# Typing Meteor Defense ☄️

A simple browser-based typing game where players must type falling words before they reach the ground.

## Technologies Used
- HTML
- CSS
- JavaScript
- Git & GitHub

## How to Play
1. A word will fall from the top of the screen
2. Type the word correctly
3. Gain points for correct words
4. Lose a life if a word reaches the bottom
5. Game ends when lives reach zero
